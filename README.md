# 迷宫冒险游戏

一个可爱的迷宫冒险游戏，支持键盘和触摸操作！

## 游戏特点

- 🎮 三种难度等级（简单/中等/困难）
- 👾 怪物追逐机制
- 🎯 收集金币、宝石和爱心
- 🏆 本地排行榜系统
- 🎨 三种主题配色
- 📱 手机触摸支持
- ⏱️ 计时和步数统计

## 快速分享方案

### 方案一：使用 GitHub Pages（推荐）

1. 访问 https://github.com/2120822671/maze-adventure/settings/pages
2. 选择 "Deploy from a branch"
3. 分支选择 "master"，文件夹选择 "/ (root)"
4. 保存后几分钟内，您的网站就会上线：
   **https://2120822671.github.io/maze-adventure/**

### 方案二：使用 CodePen

1. 访问 https://codepen.io/pen
2. 复制 maze-adventure.html 的内容进去
3. 保存即可获得分享链接

## 操作说明

- **键盘：** 方向键 或 WASD
- **手机：** 触摸滑动
- **暂停：** 空格键
